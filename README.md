# Phone Khmer Shop Website

A modern, responsive e-commerce website for "Phone Khmer Shop" - a premium smartphone retailer in Cambodia.

## Features

- **Modern & Responsive Design**: Works on all devices from mobile to desktop
- **Complete E-commerce Functionality**: Product browsing, cart, checkout
- **Telegram Bot Integration**: Order notifications and customer support
- **Multi-page Website**: Home, About, Products, Order, Contact pages
- **Product Management**: Filtering, search, detailed product information
- **Shopping Cart**: Add/remove items, update quantities
- **Order Processing**: Multi-step checkout with form validation
- **Contact Form**: With validation and Telegram notifications

## Pages

1. **Home**: Hero section, featured products, features, Telegram integration
2. **About**: Company story, mission/vision, values, team members
3. **Products**: Product grid with filtering and search functionality
4. **Order**: Shopping cart, checkout process, order confirmation
5. **Contact**: Contact form, store location, FAQ section
